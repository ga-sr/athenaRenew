## athenaRenew

A python script to auto renew all book loans.

## Instructions

1) Criar uma virtualenv

```bash
$ python -m venv venv
```

2) Entrar na virtualenv

```bash
$ source ./venv/bin/activate
```

3) Instalar dependencias

```bash
$ pip install -r requirements.txt
```
4) Edite o script
```bash
user = '12345678910'
pw = '1234'
```
5) Execute o script
```bash
$ python athena.py
```
